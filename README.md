### Installing `stow`

```sh
brew install stow
```

### Using `stow`

Passing `target` to `stow` command _(default is parent of stow dir)_

```sh
stow -t $HOME .
```

### Installing apps

```sh
brew install --cask nikitabobko/tap/aerospace
brew install --cask ghostty
brew install --cask font-iosevka-nerd-font
brew install eza

brew install neovim
brew install ripgrep
brew install fd
```
